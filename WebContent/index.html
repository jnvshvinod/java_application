Infosys CIS Hackathon 2019

Hack #3 : Team G4

Application Migration from VM to Containers Platform

Solution uses below tools:

  • GIT as Source Control Management
  • Jenkins as Integration tool
  • Tomcat/Httpd as Web Server
  • Ansible for Deployment Orchestration
  • Artifactory for Archival Storage
  • Terraform for Infrastructure Automation
  • AWS to Host Infrastructure
